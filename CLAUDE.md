# Machol Services Project Rules

## Package Manager
- **Always use `pnpm`** for package management
- Do not use npm or yarn

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React for icons
- Deploy to Vercel

## Brand Colors
- Primary Blue: #0080C6
- Dark Blue: #002A5C
- Yellow: #FFC20E
